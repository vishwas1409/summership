# Hackathon Problem Statement Selection Website

This project is now a deployable Node.js + Express + MySQL application designed for Hostinger.

## Features

- Public participant portal with search, domain filters, and live capacity tracking
- Team registration flow with server-side validation
- Maximum 5 teams per problem statement, enforced on the server
- Admin login page with password hashing and session-based authentication
- Admin dashboard to upload, edit, hide, and delete problem statements
- MySQL-backed storage for admins, problems, selections, and sessions
